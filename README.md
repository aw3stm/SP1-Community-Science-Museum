## Description
This project was part of the Semester project course at Noroff.
Based on the project brief, the focus was on mastering the full workflow of a web developer:

- Design-to-Code: Successfully translated high-fidelity wireframes and prototypes from Figma into clean, semantic HTML and CSS.

- Universal Design (Accessibility): Developed the site in accordance with regulations and guidelines for accessibility (WCAG), ensuring the content is perceivable and usable for everyone.

- Interdisciplinary Workflow: Gained experience in professional communication, version control, and collaborative problem-solving within a team setting.

- Quality Assurance: Followed industry-standard regulations and quality requirements to deliver a production-ready web solution.

- Process Documentation: Studied and documented the entire design process, identifying potential issues and implementing effective solutions to maintain project timelines.



## Built With

Design: Figma

Frontend: HTML5, CSS3 (Responsive Design)

Workflow: GitHub (Collaborative branching and PRs)

## Getting Started

### Installing

1. Clone the repo:

```bash
git clone https://github.com/aw3stm/SP1-Community-Science-Museum.git
```

2. Install the dependencies:

```
npm install
```

### Running
To run the app, run the following commands:

```bash
npm run start
```
